* `countNSXCores.json`
    * Calculate the sum of ESX cores (at levels like Datacenter, Cluster etc.), based on a Custom Property `CustomProperty|tk-isNSXenabled` that is added for example through membership in a custom group with criteria like "ESX related to NSX Transport Zone". A Custom Group definition suitable for this Super Metric can be found here: https://github.com/tkopton/aria-operations-content/blob/main/CustomGroups/tk-NSX-Enabled-ESX-Hosts.json  Important is the string in the condition that might need to be adjusted to a specific environment.
